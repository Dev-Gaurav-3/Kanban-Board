# Kanban Board 🗂️

## 📌 Description

A web-based Kanban Board application to manage tasks across different stages:
To-Do, In Progress, and Done.

## 🚀 Features

* Drag-and-drop task management
* Create tasks using modal input
* Dynamic task updates using DOM manipulation
* Task count tracking per column
* Responsive UI using Tailwind CSS

## 🛠️ Tech Stack

* HTML
* CSS
* Tailwind CSS
* JavaScript

## ▶️ How to Run Locally

1. Clone the repository
2. Open `index.html` in your browser
